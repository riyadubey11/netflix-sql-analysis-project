# Netflix SQL Analysis Project

## Overview

This project analyzes the Netflix Movies and TV Shows dataset using SQL and MySQL. The goal is to extract meaningful business insights from Netflix's content library by exploring trends, country contributions, release patterns, and content distribution.

## Objectives

* Analyze Movies vs TV Shows distribution
* Study content growth over the years
* Identify top contributing countries
* Categorize content by release periods
* Compare country-wise Movie and TV Show production
* Apply intermediate and advanced SQL techniques

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* Aggregate Functions
* CASE WHEN
* HAVING
* Subqueries
* CTEs
* Window Functions
* ROW_NUMBER
* DENSE_RANK
* LAG

* MySQL
