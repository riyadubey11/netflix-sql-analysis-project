# Netflix SQL Analysis Project

## Project Overview

Analyzed Netflix content data using SQL to identify trends, country contributions, content growth, and content distribution.

## Dataset

Netflix Movies and TV Shows Dataset (Kaggle)

## Tools Used

- MySQL
- SQL
- GitHub

## Skills Demonstrated

- GROUP BY
- Aggregate Functions
- CASE WHEN
- Subqueries
- CTEs
- Window Functions
- Ranking Functions
- Date Functions

## Business Questions Solved

1. What percentage of Netflix content is Movies vs TV Shows?
2. Categorize content by release period.
3. Find countries with more than 100 titles.
4. Which year had the biggest growth compared to the previous year?
5. Which years had below-average content production?
6. Which countries contribute more than average?
7. Find content older than 30 years.
8. Rank every title within its type.
9. Find the top 3 countries every year.
10. Which countries rely mostly on Movies instead of TV Shows?

## Key Insights

- Movies dominate the Netflix catalog.
- A few countries contribute a large portion of content.
- Content production increased significantly during certain years.
- Movie and TV Show distributions vary greatly across countries.
